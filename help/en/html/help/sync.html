Synchronizing with a PC

Run the Qtopia Desktop application on your desktop PC.
- Make sure your desktop PC is allowed to connect to the device. see the Security Application for more
- Place the Qtopia device in its cradle.
- Qtopia Desktop will automatically establish a connection to the Qtopia device.
- Click the Sync All button on the Qtopia Desktop toolbar.

Finer control is available for synchronizing individual applications, and installing software. See the Qtopia Desktop online help for more information.
