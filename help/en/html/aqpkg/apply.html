AQPkg: Apply changes

Once all desired packages have been selected, this option will either install, update or remove each package. AQPkg will prompt for whether a package already installed should be removed or upgraded. After selecting this option from the menu or toolbar, press the Start button to begin.

See Configuration for options to ignore dependencies, force reinstall, etc.

For people familiar with the command-line utility ipkg, this is equivilent to running: ipkg install or ipkg remove
