Opie Console: Getting started

First make a new profile. Open the profile dialog by clicking on the ( ) icon or chose via the profile menu.
When the dialog is opend, click the add button.

Profile tab

Profile name - Select a name under which this profile should appear in menus in the Opie Console Connection - which kind of connection should this profile have. Currently supported are:

Serial connection
Console - a normal local terminal
Infrared
modem
bluetooth

Then select a terminal type: Terminal

Afterwards continue on the

Connection tab

Depending on which connection is choosen slightly different options are shown.

Finally proceed to the

Terminal tab

when finished click on ( ) to save the profile or ( ) to cancel the profile creation. Then create another profile or exit the profile dialog by pressing ( ).

Now go to the Connection menu and select New from profile and then the created profile. This opens a tab in the main view for that connection. Now click on Connect in the Connection menu. Now you should be connected.
