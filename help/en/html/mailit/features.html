Mail It features

  • Receiving mail via POP3
  • ReplyTo, Reply to all and forwarding
  • Carbon Copies (CC:)
  • Sending mail via SMTP
  • Queuing of mail
  • Size filter
  • Download on demand
  • Mail server synchronisation
  • OPIE document support for Attachments
  • OPIE addressbook interoperation
  • Multiple mail accounts
