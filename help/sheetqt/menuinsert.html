Insert Menu

One can insert new elements to the sheet and functions to the data editor by using the items of this menu.

  • Function: Inserts names of supported functions including some mathematical and statistical functions into the data editor.
  • Cells: Inserts cells to the current sheet.
  • Rows: Inserts rows to the current sheet.
  • Columns: Inserts columns to the current sheet.
  • Sheets: Inserts sheets to the current document.
