SystemTime

Set system clock

This application sets the device's date, time and Opie global display options. The date and time can be set manually, set using an NTP time server or predicted based on previous time drift information.

This application can be accessed by either the icon on the Settings tab or by clicking on the clock in the lower right-hand corner of the screen.
