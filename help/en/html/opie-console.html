Opie Console is a Terminal Emulator Application. It supports VT100,VT102, ANSI, Linux-Console and Xterm emulation. On top of the Terminal Emulation it can be used as a locale legacy console. On top of that it supports sending and receiving of files via X-Modem, Y-Modem and Z-Modem.
