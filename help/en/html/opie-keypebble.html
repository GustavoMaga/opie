Keypebble

OPIE VNC Viewer


The VNC Interface

Keypebble is OPIE's VNC Viewer, and can connect to any server that supports the RFB protocol, including the VNC servers available at htttp://www.uk.research.att.com and http://www.tightvnc.com. You can bookmark servers you commonly connect to and can scale the remote display to fit on your current screen.

When viewing a remote screen, you can access the following actions by clicking the arrow button at the lower right hand corner of the screen:

  • Send a Control Alt Delete key stroke.
  • Resize the window to use the full screen
  • Disconnect from the server

Options

Request 8 bit Session: Limit your connection to using 8 bit color, regardless of the color depth of the remove server.

Request Shared Session: Informs the server that you are willing to share the server with other users.

Scaling Factor: Setting the scaling factor to a higher value will fit more of the remote screen onto your local display at a time
