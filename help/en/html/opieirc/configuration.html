OpieIRC: Configuration

In the IRC Menu, tap on Settings. The configurable parts are seperated into several categories:

General

Lines Displayed - Maximum amount of text to be shown inside a chat window. If there are more lines, the older ones are "forgotten". Set to 0 to deactivate this feature. (Be warned though, the memory consumption will steadily rise)

Colors

Here you can set the colors of different types of text inside OpieIRC
