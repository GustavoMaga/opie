Format Menu

One can format cells, rows, and columns in the current sheet and sheets in the current document by using the items of this menu.

  • Cells: Changes the format of the current cell selection.
  • Row -> Height: Changes the height of the rows in the selection.
  • Row -> Adjust: Adjusts the height of the rows in the selection automatically.
  • Row -> Show: Shows the hidden rows in the selection.
  • Row -> Hide: Hides all rows in the selection.
  • Column -> Width: Changes the width of the columns in the selection.
  • Column -> Adjust: Adjusts the width of the columns in the selection automatically.
  • Column -> Show: Shows the hidden columns in the selection.
  • Column -> Hide: Hides all columns in the selection.
  • Sheet -> Rename: Renames the current sheet.
  • Sheet -> Remove: Removes the current sheet from the document.
