Beaming

Some applications are beam-enabled. This means that an item of information can be sent to another beam-enabled device, wirelessly using infra-red or bluetooth.

Tap the item you want to beam to another device, point your OPIE at the other device, then tap to beam the item across. A window will now appear. The file is directly send to reachable IrDa-Devices and will query for Bluetooth enabled devices. The sending can be started by clicking on the device. If done click the X button in the decoration.
