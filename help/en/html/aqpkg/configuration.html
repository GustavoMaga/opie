AQPkg: Configuration

This option allows AQPkg to be configured. It allows modification of the feed servers, destinations on the device and proxy settings.

Servers

This tab allows the adding, modifying and removal of feed servers. To add a server, click the New button, enter the feed information and click the Update button. To modify an existing server, select it from the list, modify the information as necessary and then click the Update button. To remove an existing server, select it from the list and then click the Delete button.

Destinations

This tab allows the adding, modifying and removal of destinations.  This allows packages to be installed in different locations on the device (such as Flash memory, CF or MMC card). To add a destination, click the New button, enter the destination information and click the Updatebutton. To modify an existing destination, select it from the list, modify the information as necessary and then click the Update button. To remove an existing destination, select it from the list and then click the Delete button.

Proxy

This tab allows the setting of proxy information. This information is sometime necessary to allow AQPkg to connect to the internet when necessary. :Enter the appropriate proxy server information and click Update to save this information.
