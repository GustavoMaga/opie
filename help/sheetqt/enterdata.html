Entering Formulas

For example, to calculate average of the cells in the range A1:B3, you must enter the data as '=AVG(A1,B3)' (without quotes). And it can be done without using any keyboard by using the following procedure:
  1. Click the cell, which will show the result.
  2. Click the button.
  3. Select Average function from the Statistical menu, which will appear when you click the button.
  4. Activate the Cell Selector by clicking the button, press on the cell A1 and release on the cell B3.
  5. Click the button.
  6. Click the button.
