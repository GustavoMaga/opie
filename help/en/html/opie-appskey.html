Opie-Appskey

Opie Application Key is a program to select which buttons on your Zaurus, or Ipaq starts which programs.
The icons should match which button it controls.
It scans the /opt/QtPalmtop/apps directory for executables.
Appskey does not really remap the buttons, and can only execute binaries in the .desktop files in /opt/QtPalmtop/apps directory.
