Reading mail

The tool bar gives you several options:

Previous mail
Read the previous mail in the inbox/outbox list
Next mail
Read the next mail in the inbox/outbox list

ASCII/Rich text
Switch between standard text (with ASCII line breaks) and HTML/RichText view (needs HTML line breaks).

Attachment available
Click on the icon to open the attachment view.

Reply to the current mail on a single click.
If you click and hold this icon, a popup menu is opened with several options:
Reply to all recipients
Forward the mail

Remove mail

Removes the current mail

Mail download

Completely fetches a mail if is not yet downloaded. This icons only appears if needed.
