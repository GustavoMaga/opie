Opie Console is Terminal Emulator Application. It supports VT100,VT102, ANSI, LinuxConsole and XTerm emulation. On top of the Terminal Emulation it can be used as a locale legacy console client. On top of that it supports sending and receiving of files via X-Modem, Y-Modem and Z-Modem.
