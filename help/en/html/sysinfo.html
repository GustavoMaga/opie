SysInfo

System information viewer

This application presents a set of tabbed pages each containing information about the current status of the device's Memory, Storage, CPU, Processes, Kernal Modules, and software Versions.

Memory
Displays memory use. Used memory is displayed on the left of the bar in red. Total available memory equals Free plus Cached memory. Opie makes use of spare Free memory as Cache to improve speed, but will reclaim Cached memory whenever necessary.

Storage
Displays storage available for any media currently in use.

CPU
Displays the CPU usage, or how hard the processor is working.

Processes
Displays all processes currently running on the system.  Additional information about the process can be viewed by clicking and holding the desired process. Signals can be sent to a process by selecting the process, selecting the desired signal and then clicking the Send button.

WARNING: Stopping processes can render the device unstable until it is rebooted.

Kernel Modules
Displays all kernel modules currently loaded on the system.  Additional information about the module can be viewed by clicking and holding the desired module. A module can be removed by selecting the process, selecting the desired removal option and then clicking the Send button.

WARNING: Removing modules can render the device unstable until it is rebooted.

Version
Displays software version numbers.
