The Game menu
  • new
    start a new game
  • load
    load a previusly saved game here
  • save
    save the current game.
    The game will be named automatically with the current date and time
  • delete
    delete a saved game
top
The Theme menu
  • new
    create a new theme or alter an existing one based on the componts save on your z.
    make sure to give it a unique name, so you do not overwrite an existing theme !
  • load
    load a theme
  • save
    save the current theme
  • default
    make the current theme the default theme (the one that gets loaded when you start the game)
  • delete
    delete a theme (the theme with the name default can not be deleted)
