Account settings

Chosing either New or a specific account from the Account menut opens the Account settings dialog
Here you can fill out several fields:
  • Account name A descriptive name as shown in the account menus

    Your name Your name to be used as the From: name

    EMail address Your email address to be used as the From: address

    POP user name Your login name on the POP3 server

    POP password Your password on the POP3 server

    POP server The POP3 mailserver to get mail from

    SMTP server The SMTP mailserver to send mail to

If the synchronize checkbox is active only mail will be downloaded which is not in the inbox yet.
The mail size spinbox defines the maximum mai size to fetch, mails larger than this limit will only be fetched as headers.

Click the ok button in the upper right corner to accept or the cancel icon beneath it to abort.
