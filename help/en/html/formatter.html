Opie Formatter

is a disk formatting front end. You can also perform disk fixing operations.
It does not use fdisk. It does not perform partioning.
If you are going to reformat anything to another filesystem, I recommend running fdisk and toggling the partition type, and reboot prior to formatting.
It scans all available disks, and filesystem supported.

On the Main tab

You select which disk to format, and select which file system to format it too, push the format button, and hope it works. ;)
Please back up any important data before you format it.

On the Advanced tab

You can edit fdisk in textedit, or change the mount point with the mount point edit box.
When you select which dev, you will see fiole system infos, Type of disk, what it is formatted with.
It also shows the total space, used space, and space available on the selected device.

Pushing the check disk button will perform a file system check, and output so you can see the changes it performs.

Please backup any important data before you use this program!
