confedit

A big fat Warning:

Do NOT use confedit!

unless you know what you are doing...

Possible Actions

By highlighting an item a editor will show up.
After pressing on a item for some time a popup menu with the possible actions will show up.

conf file

  • Save:
    save the changes
  • Revert:
    revert the file to the backup copy
    i.e. undo the last change
  • Delete:
    delete the file from the filesystem
    This cannot be undone, your settings will be lost and reverted to the default!

group

  • Revert:
    revert the group to the saved version
    i.e. undo the last change
  • Delete:
    delete the group
    This cannot be undone, your settings will be lost and reverted to the default!

entry

  • Revert:
    revert the entry to the saved version
    i.e. undo the last change
  • Delete:
    delete the entry
    This cannot be undone, your settings will be lost and reverted to the default!
